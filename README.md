# Calculadora
  É uma Calculadora bem simples com funções: de adição, multiplicação,subtração e divisão.

  **como usar?**
  
    muito simples, basta cliclar nos números
    
    escolher uma das funções e clicar em "=" 
    
    e sua conta estará feita!!
